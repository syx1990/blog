什么是ssh？

SSH 为 Secure Shell 的缩写，由 IETF 的网络小组（Network Working Group）所制定；SSH 为建立在应用层基础上的安全协议。SSH 是目前较可靠，专为 远程登录 会话和其他网络服务提供安全性的协议。

如何快速的配置ssh免密登录

配置ssh分三步走

  1. 在本地创建公私钥
  2. 用ssh-copy-id 将公钥复制或传输到远程服务器,并将身份标识文件追加到服务器的
  3. 配置本机ssh config文件
  • 修改服务器 /etc/ssh/sshd_config,并重启sshd服务 愉快的玩耍吧。
